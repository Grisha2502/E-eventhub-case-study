# E-EventHub – eSports Platform (Case Study)
Product case study for an eSports platform for Nepali audience
## Overview
This project is a product case study focused on building a localized eSports platform for the Nepali audience.
## Problem
- Lack of centralized platform for eSports in Nepal
- Difficulty discovering tournaments and events
## Solution
A platform connecting players, organizers, and tournaments in one place.
## Key Features
- Tournament listings
- User profiles
- Event registration
- Notifications
## Design
Wireframes and UI designed using Figma.
